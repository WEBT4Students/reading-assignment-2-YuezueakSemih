What are the most essential technologies that made the World Wide Web such a success? IP-protocol

How are resources (html pages, images, etc) addressed in the web? A URL

What are the general components of a url? the scheme, subdomain, top-level domain, second-level domain, and subdirectory.

What is a web server? web server is computer software and underlying hardware that accepts requests via HTTP (the network protocol created to distribute web content) or its secure variant HTTPS.

What does a web server? A web server connects to the Internet and supports physical data interchange with other devices connected to the web

How can content be stored on a web server? They often aren't stored directly, but generated dynamically by server-side code and possibly cached by a reverse proxy.

How is content in webpages described? Page content refers to all the information contained in a website. Page content can be displayed as text, links, images, audio, animation or videos among other things

What is a web client? A web client is a client-side application used for connecting to a web server over HTTP

What does a web client? A web client lets the user request something on the server, and shows the user the result of the request.

Is Web Client and Web Browser is the same? No

What is the html head element? The "head" element is a container for metadata (data about data) and is placed between the "html" tag and the "body" tag.

What does "!DOCTYPE` html" mean? "!DOCTYPE html" is the doctype for the document. It basically tells the web browser how to render the page (there are different doctypes).

What are the two top most elements within the html element? "head" and "body"

How to emphasize a text and how is it displayed in the browser? "em" tag

How to mark up important text and how is it displayed in the browser? "mark"

Identify the difference between serif and sans-serif fonts

 A serif is a decorative stroke that finishes off the end of a letters stem (sometimes also called the feet of the letters). In turn, a serif font is a font that has serifs, while a sans serif is a font that does not (hence the sans).
